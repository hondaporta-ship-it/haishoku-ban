# 配色番 / Haishokuban

ATSのシフト表Excelに、取引先ごとの色を自動で付ける社内ツール。

## 何をするもの

毎週カシサコさんが手で塗っているシフト表の色付けを自動化する。
**「同じ取引先で複数現場あるもの」だけに色を付けて、現場ごとに必ず違う色**にする。
隊員が現場を取り違える事故を防ぐための、安全装置としてのツール。

## 使い方（カシサコさん向け）

1. URLをブラウザで開く
2. シフト表のExcelを画面の枠にドラッグ&ドロップ
3. **全部の月に自動で色がつく**
4. プレビューしたい月を選んで結果を確認
5. **「ダウンロード」を押す**
6. ダウンロードされたExcelをLINE WORKSに貼る

## v0.7 の変更点（大きな作り直し）

### ExcelJSを廃止、JSZip直接書き換え方式に転換
v0.6まで使っていたExcelJSは、ファイルを読み書きする際に**グラフ・数式の計算チェーン・コメント・外部リンクなどを大量に欠落**させていた（元3.2MB→2.0MB）。これがExcelを開く時の「修復しますか」エラーの原因だった。

v0.7では、元ファイルを1バイトも壊さずに保持し、**色のセルだけをXMLレベルで直接書き換える**方式（JSZip）に変更。グラフも数式も全部残る。

### 色ロジックの刷新
- **色相を取引先数で360度を等分割り当て**：取引先数ぶんの色相を必ず確保する
- **同じ取引先内は明度を等間隔分割**：現場が何個あっても全部違う明度になる
- 結果として、**全現場が数学的に必ずユニークな色**になる

## 仕組み

- 単一の `index.html` で完結
- 外部APIなし。Excelファイルはブラウザ内で処理。サーバーには一切送信されない
- ライブラリ：JSZip（XML直接操作）、FileSaver.js
- ユーザー追加マッピングは `localStorage` で端末保存
