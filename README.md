# 配色番 / Haishokuban

ATSのシフト表Excelに、取引先ごとの色を自動で付ける社内ツール。

## 何をするもの

毎週カシサコさんが手で塗っているシフト表の色付けを自動化する。
**「同じ会社で複数現場あるもの」だけに色を付けて、現場ごとに濃淡で区別**する。
隊員が現場を取り違える事故を防ぐためのツール。山脇さんからの「見にくい」というクレームが発端だが、本質は「現場ごとに違う場所だから、絶対に間違えてはいけない」という安全装置。

## 使い方（カシサコさん向け）

1. URLをブラウザで開く
2. シフト表のExcelを画面の枠にドラッグ&ドロップ
3. 月を選ぶ（最新月が自動で選ばれる）
4. 各現場の取引先名を入力（初回のみ。次回からブラウザが記憶）
5. 「色を付ける」→「ダウンロード」
6. ダウンロードされたExcelをLINE WORKSに貼る

## 仕組み

- 単一の `index.html` で完結
- 外部APIなし。Excelファイルはブラウザ内で処理され、サーバーには一切送信されない
- ライブラリ：ExcelJS（セル色塗り）、FileSaver.js（ダウンロード）
- 取引先マッピングは `localStorage` で端末保存

## 色のロジック

- 1社で1現場 → 色なし
- 1社で複数現場 → 取引先ごとに色相を割り当て、現場ごとに濃淡（4段階）で区別
- 取引先数が10を超えたら色相を循環
