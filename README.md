# 配色番 / Haishokuban

ATSのシフト表Excelに、取引先ごとの色を自動で付ける社内ツール。

## 何をするもの

毎週カシサコさんが手で塗っているシフト表の色付けを自動化する。
**「同じ会社で複数現場あるもの」だけに色を付けて、現場ごとに濃淡で区別**する。
隊員が現場を取り違える事故を防ぐためのツール。

## 使い方（カシサコさん向け）

1. URLをブラウザで開く
2. シフト表のExcelを画面の枠にドラッグ&ドロップ
3. **全部の月に自動で色がつく**
4. プレビューしたい月を選んで結果を確認
5. **「ダウンロード」を押す**
6. ダウンロードされたExcelをLINE WORKSに貼る

## v0.6 の変更点

- **初期表示を今日の月に**：開いた時のプレビューが最終月（2027年3月）から始まり毎回逆行していたのを、今日の月（なければ現在月以降の最寄り、それもなければ最古月）を初期表示にするよう変更
- **全月一括色付け**：ファイルをドロップした瞬間、全月別シートに色を付ける。月の切り替えはプレビュー表示用のみ
- **色クリアの完全リセット**：`cell.fill = null` で既存色を完全削除。テーマカラーが消しきれず「修復しますか」エラーが出ていたのを修正

## 仕組み

- 単一の `index.html` で完結
- 外部APIなし。Excelファイルはブラウザ内で処理。サーバーには一切送信されない
- ライブラリ：ExcelJS、FileSaver.js
- ユーザー追加マッピングは `localStorage` で端末保存
