# 配色番 / Haishokuban

ATSのシフト表Excelに、取引先ごとの色を自動で付ける社内ツール。

## 何をするもの

毎週カシサコさんが手で塗っているシフト表の色付けを自動化する。
**「同じ会社で複数現場あるもの」だけに色を付けて、現場ごとに濃淡で区別**する。
隊員が現場を取り違える事故を防ぐためのツール。

## 使い方（カシサコさん向け）

1. URLをブラウザで開く
2. シフト表のExcelを画面の枠にドラッグ&ドロップ
3. 月を選ぶ（最新月が自動で選ばれる）
4. 色がついた結果がすぐ表示される
5. **「ダウンロード」を押す**
6. ダウンロードされたExcelをLINE WORKSに貼る

主要な取引先はアプリに最初から内蔵されているので、ファイルを入れるだけで色が付く。

## v0.4 の変更点

- **色残りバグ修正**：休み（「-」）や内勤、研修のセルに、元Excelの手作業塗り色（緑色）が残っていたのを修正。配色番が処理する範囲は、マッピングされた現場以外はすべて色をクリアする

## v0.3 の変更点

- ダウンロードエラーの修正：元Excelの`A10:XFD10`列全体データ入力規則による `16385 is out of bounds` エラーを修正

## v0.2 の変更点

- プリセット同梱で手入力ゼロ
- 即時プレビュー（「色を付ける」ボタン廃止）
- 表記揺れ対応（半角/全角の正規化）

## 仕組み

- 単一の `index.html` で完結
- 外部APIなし。Excelファイルはブラウザ内で処理。サーバーには一切送信されない
- ライブラリ：ExcelJS、FileSaver.js
- ユーザー追加マッピングは `localStorage` で端末保存

## 既知の課題

- ダウンロードしたExcelを開くと「修復済み」と表示されることがある。ExcelJSが書き出すxlsxの軽微な不整合をExcelが自動修復した跡で、ファイルを開いて編集することはできる。実害は今のところ確認されていない。
