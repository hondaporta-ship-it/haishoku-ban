# 配色番 / Haishokuban

ATSのシフト表Excelに、取引先ごとの色を自動で付ける社内ツール。

## 何をするもの

毎週カシサコさんが手で塗っているシフト表の色付けを自動化する。
**「同じ会社で複数現場あるもの」だけに色を付けて、現場ごとに濃淡で区別**する。
隊員が現場を取り違える事故を防ぐためのツール。

## 使い方（カシサコさん向け）

1. URLをブラウザで開く
2. シフト表のExcelを画面の枠にドラッグ&ドロップ
3. 月を選ぶ（最新月が自動で選ばれる）
4. **色がついた結果がすぐ表示される → 「ダウンロード」を押す**
5. ダウンロードされたExcelをLINE WORKSに貼る

**入力は不要**。アプリの中に主要な取引先（アスミオ、中村、ホンダ、九州、佐伯、オークス、オカトクなど）の情報が最初から入っているので、ファイルを入れるだけで色が付く。

新しい現場が出てきた時だけ、画面に「⚠ 未登録の現場があります」と出るので、その時だけ取引先名を入れる（次回からは自動で覚える）。

## v0.2 の変更点

- **プリセット同梱**：主要取引先と現場の対応を内蔵。手入力ゼロで動く
- **即時プレビュー**：「色を付ける」ボタンを廃止。ファイル選択 → 月選択で即座に結果が出る
- **未登録警告**：プリセットにない現場だけ、未登録警告として表示
- **色相循環の警告**：取引先が10社を超えたら警告
- **トースト通知**：`alert()` を廃止し、画面右上のトーストに変更
- **表記揺れ対応**：半角カナ/全角カナ、半角/全角数字を正規化してマッチング

## 仕組み

- 単一の `index.html` で完結
- 外部APIなし。Excelファイルはブラウザ内で処理され、サーバーには一切送信されない
- ライブラリ：ExcelJS（セル色塗り）、FileSaver.js（ダウンロード）
- 取引先マッピング（ユーザー追加分）は `localStorage` で端末保存

## 色のロジック

- 1社で1現場 → 色なし
- 1社で複数現場 → 取引先ごとに色相を割り当て、現場ごとに濃淡（4段階）で区別
- 取引先数が10を超えたら色相を循環（警告を表示）
