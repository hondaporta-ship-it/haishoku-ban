# 配色番 / Haishokuban — Claude Code 向けメモ

## リポジトリ

- GitHub: `hondaporta-ship-it/haishoku-ban`
- Pages: `https://hondaporta-ship-it.github.io/haishoku-ban/`

## 構成

単一 HTML ファイル（`index.html`）+ `README.md` のみ。ビルド不要。

## デプロイ手順

1. `git pull`
2. `index.html` / `README.md` を上書き
3. `git add . && git commit -m "vX.X: 変更内容"` && `git push`
4. Pages 反映確認（1〜2分）

## 注意事項

- `index.html` の `PRESET_RAW` は樫迫さんの確認を経て随時修正
- 推測ベースのマッピング（クライムアップ・サンガード・ユリシス・アンシェル・興和・カシワ）は要確認
- `STORAGE_KEY` を変えると既存ユーザーのマッピングが消えるので慎重に
